## 2. 核心设计原则(量化专属,区别于普通 ML)

这是本 spec 最重要的部分——普通 ML 教程直接 `train_test_split` 随机切分,**在量化里是致命错误**。

1. **时间序列切分,严禁随机切分**
   - 面板数据有时间维度,随机切分会造成**未来信息泄露**(用未来预测过去)。
   - 方案:按报告期切分。**前 3~4 期训练 → 后 1~2 期测试**。
   - 推荐:训练 = 2021Q2/Q3/Q4,验证 = 2022Q1,测试 = 2022Q2(样本外)。

2. **横截面标准化(Cross-sectional),按报告期分组做**
   - 因子应在**每个报告期内部**做标准化/去极值,而非全样本一起做。
   - 原因:不同时点市场整体估值水平不同,跨期混合标准化会引入偏差。

3. **去极值(Winsorize)**
   - PE/PCF 等存在极端值,用**分位数截尾(1%/99%)** 或 **MAD(中位数绝对偏差)** 处理。

4. **类别不平衡(40/60)**
   - 评价**别只看 Accuracy**,优先 **AUC / F1 / Precision / Recall**。
   - 模型层用 `class_weight='balanced'` 或阈值调整。

5. **评价要落到"选股/交易"语义**
   - 分类指标之外,补一个**按预测概率分层的简易回测**:看高分组 vs 低分组的标签命中率差异(多空区分度),呼应 IC/单调性思想。

## 3. Notebook 结构规划(章节 = cell 分块)

> 文件名建议:`stock_ml_classification_demo.ipynb`

### 第 0 部分 · 环境与配置
- 导入库:pandas、numpy、scikit-learn、matplotlib、seaborn;可选 xgboost/lightgbm。
- 全局配置:随机种子、中文字体、A 股红涨绿跌配色常量、绘图风格。

### 第 1 部分 · 数据加载与认识(EDA)
- 相对路径读入 CSV,`Date` 转 datetime,列名做安全重命名(处理含逗号/括号的中文列名 → 英文短名映射表)。
- 基础画像:形状、dtypes、标签分布(饼图)、各报告期样本量(柱状图)。
- 特征分布:直方图/箱线图看离群、`describe()`、缺失确认。
- 相关性热力图:看因子间共线性(PE 与 PE扣非 高度相关等)。

### 第 2 部分 · 数据预处理(量化流水线)
- **去极值**:按报告期分组,对 17 个因子做 1%/99% 分位截尾。
- **标准化**:按报告期分组做 z-score(横截面标准化)。
- **标签编码**:Y → 1/0。
- 封装成可复用函数,便于对训练/测试分别 fit-transform(标准化参数只用训练期拟合)。

### 第 3 部分 · 数据集切分(时间序列)
- 按报告期切:train(前3期)/ valid(第4期)/ test(第5期,样本外)。
- 打印各集合样本量与正例比例,确认分布一致。
- 明确 `X_train/y_train ... X_test/y_test`。

### 第 4 部分 · 模型开发(逐个建模 + 统一接口)
统一封装 `train_eval(model, name)`:训练 → 预测 → 输出概率 → 记录指标。依次实现:

| # | 模型 | 库 | 演示重点 |
|---|---|---|---|
| 1 | 逻辑回归 LR | sklearn | 基线、系数可解释(哪些因子正/负向) |
| 2 | 决策树 DT | sklearn | 可视化树结构、特征重要性、过拟合演示 |
| 3 | 随机森林 RF | sklearn | 集成降方差、特征重要性排序 |
| 4 | 梯度提升 GBDT/XGBoost | sklearn/xgboost | 表格数据高精度对照 |
| 5 | KNN | sklearn | 距离类模型、需标准化的直观体现 |
| 6 | SVM | sklearn | 最大间隔(可选,数据量大时注意耗时) |
| 7 | 朴素贝叶斯 | sklearn | 快速基线对照 |

- 每个模型:默认参数先跑通,再对 RF/XGBoost 用验证集做简单调参(GridSearch/手动)。

### 第 5 部分 · 模型评价与对比(呼应评价指标手册)
- 对每个模型在**测试集(样本外)** 输出:
  - 混淆矩阵(热力图,涨跌语义标注)
  - Accuracy / Precision / Recall / F1
  - ROC 曲线 + AUC(所有模型画在**同一张图**对比)
  - P-R 曲线(应对不平衡)
- 汇总一张**模型对比表**(各指标横向对比,高亮最优)。
- 特征重要性对比(树模型)+ LR 系数方向解读。

### 第 6 部分 · 量化视角延伸:简易分层回测
- 用最优模型输出的**预测概率**给测试期股票打分排序。
- 分 5~10 组(分位),统计每组的实际正例率 / 平均标签。
- 画分层柱状图:看是否**单调**(高分组命中率明显高于低分组 = 信号有区分力)。
- 计算 Top 组 vs Bottom 组的差异,类比多空组合的选股有效性。

### 第 7 部分 · 结论与建议
- 哪个模型综合最优(结合 AUC + F1 + 分层单调性,而非单看准确率)。
- 哪些因子最有效(重要性 + 方向)。
- 局限与改进方向:数据仅 5 期偏短、可加滚动回测(Walk-Forward)、加入行业中性化、成本敏感阈值等。

## 6. 关键陷阱清单(务必规避)

- ❌ 随机切分数据 → ✅ 按时间(报告期)切分,防未来泄露
- ❌ 全样本一起标准化 → ✅ 标准化参数只用训练期 fit;横截面按报告期分组
- ❌ 只看 Accuracy → ✅ 不平衡场景以 AUC/F1/分层单调性为主
- ❌ 忽略离群值 → ✅ 去极值(分位截尾/MAD)
- ❌ 把 PE 与 PE扣非 等强共线性因子无脑全塞进 LR → ✅ 关注共线性,必要时筛选或正则
- ⚠️ 标签 Y 的业务定义先确认,决定所有指标的解读方向
